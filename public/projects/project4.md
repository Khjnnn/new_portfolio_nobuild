# 로또 번호 생성기

React 기반으로 개발한 로또 번호 생성 및 관리 웹 애플리케이션입니다.

## 🎯 프로젝트 개요
- **목적**: 사용자가 원하는 로또 번호를 생성하고 관리할 수 있는 웹 서비스
- **플랫폼**: 웹 애플리케이션
- **배포**: Vercel

## 🔗 링크 URL
[https://lotto-omega-lyart.vercel.app/](https://lotto-omega-lyart.vercel.app/)

## ⚡ 주요 기능
- 랜덤 로또 번호 생성 (1-45 숫자 6개 조합)
- 동행복권 API 연동을 통한 실시간 최근 당첨 번호 확인
- 생성 기록 확인
- 반응형 디자인

## 🛠️ 기술 스택
- **Frontend**: React, JavaScript, HTML/CSS
- **배포**: Vercel
- **상태 관리**: React State
- **스타일링**: CSS-in-JS

## 🚀 개발 과정
1. React 프로젝트 구조 설계
2. 로또 번호 생성 알고리즘 구현
3. 동행복권 API 연동 및 데이터 파싱
4. 사용자 인터페이스 디자인
5. 실시간 당첨 정보 업데이트 기능 구현
6. 반응형 디자인 적용
7. Vercel을 통한 배포

## 💡 학습한 점
- React Hooks 활용
- 상태 관리 최적화
- 외부 API 연동 및 데이터 파싱
- 웹 배포 자동화
- 사용자 경험 개선

## 📱 특징
- 직관적인 사용자 인터페이스
- 빠른 번호 생성 속도
- 모바일 친화적 디자인
- 실시간 번호 업데이트
